    Awareness Activity

    Finally working on the first phase of project Wildbeest, which is an activity to inspire young designers to start talking more about design process instead of outcomes.

    Explored some designs for the user end.

    This activity would only be effective if it has a real time syncing feature so that people at the open studio and other events could immediately be inspired to start converasations. But I couldn’t do it with Figma, so I reached out to a computer science student, working together to make it come to life.

    And together, we decided on the UI that is most achievable for the timeline as I wanted it to be ready during open studio so I could do user testing as well as collect datas.
